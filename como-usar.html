Bienvenido a la guía de uso de SpinTrainer

Descubre cómo nuestra aplicación puede transformar tus sesiones de spinning. En esta sección, te guiaremos paso a paso para que puedas aprovechar todas las funcionalidades que SpinTrainer tiene para ofrecerte, garantizando una experiencia de entrenamiento óptima y personalizada.

Teléfonos Móviles

Registrarse

Login:

Esta parte permite a los usuarios ya registrados acceder a la aplicación mediante la introducción de un correo electrónico o nombre de usuario y una contraseña.

Registro de usuario:

Para registrarse, ingrese datos personales como nombre, apellido, número de teléfono y correo electrónico. Luego elija un nombre de usuario y contraseña. Finalmente, seleccione entre los roles de administrador o entrenador para su cuenta.

Recuperación de contraseña y usuario:

De ser necesario, la aplicación también cuenta con un respaldo para recuperar contraseña y recuperar usuario mediante un código que se enviará al correo electrónico.

Menú Principal

Inicio:

La pantalla de inicio es el primer punto de acceso al abrir la aplicación, donde el usuario puede ver una lista de rutinas guardadas y buscar rutinas específicas por nombre. Incluye una lista de todas las rutinas guardadas para fácil acceso y una barra de búsqueda en la parte superior para localizar rápidamente rutinas ingresando su nombre.

Sección de rutina:

Cuando un usuario selecciona una rutina desde la pantalla de inicio, es redirigido a esta pantalla que presenta un menú con opciones para gestionar la rutina seleccionada, incluyendo duplicar para crear una copia, modificar para editar los detalles, y eliminar para borrar la rutina.

Menú:

En la esquina superior izquierda de la aplicación, un ícono de menú despliega varias opciones de navegación y configuración al ser seleccionado, permitiendo al usuario volver al menú principal, acceder y editar su perfil, obtener ayuda a través de recursos de soporte, y cerrar sesión o salir de la aplicación.

Creación de una Sesión

Nombre de la Sesión:

En esta primera pantalla del proceso de creación de una sesión, el usuario establece los detalles básicos de la sesión, incluyendo el nombre de la sesión, la fecha y hora de inicio, y la duración.

Datos de Ejercicio:

En esta pantalla, el usuario introduce detalles específicos del ejercicio que se realizará, incluyendo el tipo de ejercicio, la posición de las manos, los valores de Rendimiento Percibido de Esfuerzo (RPM) medio y máximo, y la zona de energía en la que se trabajará.

Guardar Sesión:

En esta pantalla final del proceso de creación de una sesión, se muestra un resumen completo de todos los detalles ingresados, como el nombre, fecha, hora, duración y datos específicos del ejercicio, junto con una opción para guardar y confirmar la sesión.

Pantallas de Televisión

Selección de Perfil

Perfiles:

Los usuarios seleccionar su perfil de entrenador mostrando fotos y nombres de los entrenadores registrados en la aplicación.

PIN de Usuario

Seguridad:

Después de seleccionar un perfil, el usuario debe ingresar un PIN de usuario en esta pantalla para verificar su identidad, garantizando que solo los usuarios autorizados accedan a sus perfiles y a la información sensible almacenada en la aplicación.

Selección de Rutina

Rutinas:

En esta pantalla se muestran todas las rutinas de entrenamiento guardadas, con su nombre y fecha de ejecución . El usuario puede buscar, filtrar y seleccionar rutinas específicas para iniciar la reproducción, facilitando el acceso rápido y la gestión eficiente de los entrenamientos.

Preproducción de rutina

Iniciar:

En esta pantalla, se reproduce la rutina seleccionada mostrando indicaciones detalladas como el tipo de movimiento, posición de las manos, valores de RPM mínimo y máximo, y el tiempo de cada ejercicio. El usuario recibe instrucciones visuales y detalladas para realizar el entrenamiento correctamente, optimizando los beneficios del ejercicio y reduciendo el riesgo de lesiones.
